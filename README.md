# 📸 Clickify — Personal Photo Booth

> A beautiful, private photo booth web app with login system, cute filters, frames & stickers.

## ✨ Features

- 🔐 **Login & Signup** — Each user has their own private account
- 📸 **Photo Booth** — Take photos with your camera directly in the browser
- 🎨 **12 Filters** — Natural, Noir, Retro, Blossom, Forest, Ocean, Dream, Violet, Mist, Bold, Rose, Cyber
- 🖼️ **9 Photo Frames** — Hearts, Flowers, Stars, Rainbow, Polaroid, Vintage, Kawaii, Glitter, Garden
- ✨ **16 Stickers** — Emoji overlays added to your photos
- 📐 **5 Photo Sizes** — Square, Portrait, Wide, Standard, Story
- 🎞️ **Film Strip** — Single, Strip ×4, Burst ×3 shoot modes
- 💾 **Private Gallery** — Your photos are visible only to you
- ⬇️ **Download** — Save photos individually or as a film strip
- 🛡️ **100% Private** — No uploads, no servers, everything stays on your device

## 🔒 Privacy & Security

| Feature | Details |
|---|---|
| Photo Storage | Stored only in your browser's localStorage |
| User Isolation | Each user's photos are completely separate |
| No Server | Zero backend — no data ever leaves your device |
| No Tracking | No cookies, no analytics, no ads |
| Password | Hashed before storing — never saved as plain text |
